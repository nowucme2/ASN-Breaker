## Overview

ASN Breaker is an automated reconnaissance framework designed to enumerate infrastructure from ASN ranges and discover exposed attack surfaces.

The tool supports multiple input sources, applies intelligent filtering, safely limits scanning ranges, performs service discovery, captures screenshots, runs vulnerability scans, and generates a final HTML report.

It is useful for:

- External attack surface discovery
- Internal pentesting preparation
- Infrastructure asset discovery
- Recon automation
- Red team reconnaissance

---

## Features

• BBOT ASN table input support  
• Direct ASN lookup support  
• Direct CIDR/subnet scanning  
• CDN/cloud infrastructure filtering  
• Smart subnet intelligence engine  
• Safe scanning limits for large networks  
• Port scanning using Naabu  
• HTTP service discovery using httpx  
• Screenshot capture using Gowitness  
• Vulnerability scanning using Nuclei  
• Final HTML report generation  
• Downloadable IP inventory

---

## Supported Input Modes

### BBOT ASN Table

Use ASN table generated by BBOT.

```

python3 asn_breaker.py -b asns-table.txt

```

Example BBOT row:

```

AS8075 | 13.104.0.0/14 | MICROSOFT

```

ASN Breaker extracts the subnet automatically.

---

### ASN Lookup

Fetch ASN prefixes from BGPView API.

```

python3 asn_breaker.py -a AS13335

```

Example output:

```

104.17.144.0/20
172.64.0.0/13

```

---

### Direct CIDR Scan

```

python3 asn_breaker.py -c 192.168.1.0/24

```

---

## Subnet Intelligence Engine

Each subnet is analyzed before scanning.

| Subnet Size | Action |
|-------------|--------|
| ≤1000 IPs | Automatically accepted |
| 1000–50000 IPs | User confirmation |
| >50000 IPs | Limited to first 1000 IPs |

Example:

```

197.227.0.0/16 | 65536 IPs
Large subnet detected
Limiting scan to first 1000 IPs

```

This prevents accidental scanning of very large networks.

---

## CDN Filtering

The tool automatically skips infrastructure belonging to major CDN/cloud providers.

Filtered providers include:

- Cloudflare
- Akamai
- Fastly
- Amazon
- AWS
- Google
- Microsoft
- Azure

Example:

```

104.17.144.0/20 → Cloudflare → skipped
13.104.0.0/14 → Microsoft → skipped

```

## System Requirements

Python 3.9+

Required tools:

- Naabu
- httpx
- Gowitness
- Nuclei

Install example:

```

sudo apt install naabu gowitness

```

Install ProjectDiscovery tools:

```

go install github.com/projectdiscovery/httpx/cmd/httpx@latest
go install github.com/projectdiscovery/nuclei/v3/cmd/nuclei@latest

```

Ensure `$HOME/go/bin` is in PATH.
